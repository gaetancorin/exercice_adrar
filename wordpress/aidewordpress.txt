INSTALLER WORDPRESS=
télécharger xampp, activer apache et mysql
télécharger le zip sur site de wordpress
créer fichier dans htdoc et mettre le zip dedans
créer une bdd vide avec phpmyadmin au MEME NOM du dossier dans htdoc
lancer le site avec localhost/nomdudossier

METTRE FICHIERS WORDPRESS dans un sousdossier=
dabors indiquer l'url sur le wordpress(reglages, general, changer 
Adresse web de WordPress (URL)   et  Adresse web du site (URL)  avec la meme url
puis enregistrer,  le site crash.
on déplace tous les fichiers dans le sousdossier qu'on a créer
et voila !
